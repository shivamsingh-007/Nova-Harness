PromptAssembly — 7 Records + 3 Enums
■
PromptAssembly
assembly_id + run_id + 6 component blocks + messages
PromptAssembly
├─ assembly_id: str ● run_id: str
├─ instruction_layers: List[InstructionLayer] (≥1)
├─ context_blocks: List[ContextBlock]
├─ tool_exposures: List[ToolExposure] (unique names)
├─ constraints: List[ConstraintBlock]
├─ verification: Optional[VerificationBlock]
└─ messages: List[MessageBlock] (≥1)
Six Component Blocks
■
InstructionLayer
layer_id, type (system/global/repository/task), title, content, priority
Empty layer_id, title, or content → error
Default priority = 0
■
ContextBlock
block_id, type (file/failure/doc/summary), title, content, source_ref, priority
Empty block_id, title, or content → error
source_ref optional
■
ToolExposure
tool_name, description, input_schema_summary, requires_approval
Empty tool_name → error
Duplicate names in one assembly → error
Default: no approval required
■
ConstraintBlock
title + list of rules
Empty title → error
Empty rules list → error
■
VerificationBlock
summary + required_checks + done_definition
Must have at least one check or done_definition item
■
MessageBlock
role (system/user/assistant) + content
Empty content → error
At least one required on assembly
