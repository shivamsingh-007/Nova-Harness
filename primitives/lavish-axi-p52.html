Validation Rules
NON-BLANK graph_id, session_id, entry_node_id, node/edge/branch/join/checkpoint IDs
UNIQUE node and edge IDs must be unique within the graph
TERMINAL at least one terminal_node must exist
ENTRY entry_node_id must reference an existing node
EDGE from_node_id and to_node_id must reference existing nodes
PARENT parent_node_id must reference an existing node
BRANCH branch entry_node_id and branch_node_ids must reference existing nodes
JOIN join_node_id and incoming_node_ids must reference existing nodes
JOIN COUNT required_completion_count must not exceed incoming_node_ids count
CHECKPOINT checkpoint graph_id must match metadata graph_id
