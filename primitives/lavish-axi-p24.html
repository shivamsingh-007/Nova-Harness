Enforcement Flow
EvaluationResult created with metrics + mandatory checks ├→ QualityGatePolicy evaluated against metric results ├→ Each threshold checked: score >= minimum_score? ├→ Mandatory check failures trigger BLOCK/ESCALATE/HOLD ├→ Low confidence vs minimum_confidence prevents auto-ALLOW └→ QualityGateDecision produced: ├→ ALLOW → output proceeds ├→ RETRY → step re-executes (soft failure) ├→ HOLD → paused for review ├→ ESCALATE → human-in-the-loop review └→ BLOCK → output rejected, run halted
Validation Rules
NON-EMPTY evaluation_id, subject_id, evaluator_type, policy_id, decision_id, metric_name, check_name, evidence_id, label, content
RANGE score, threshold, aggregate_score, confidence all in [0.0, 1.0]
MANDATORY mandatory threshold action_on_fail must be BLOCK, ESCALATE, or HOLD (not ALLOW/RETRY)
POLICY thresholds not empty • metric_names unique
DECISION passed_gate must result in ALLOW • BLOCK/ESCALATE need rationale • failure must list at least one failed metric or check
