Validation Rules

WAIVER required criteria/checks waived without notes rejected at model and envelope level
ACCEPT accepted requires all required criteria met/waived and required DoD checks passed/waived
CAVEATS accepted_with_caveats must include followup_actions
REJECT rejected/needs_rework must include decision_reason
RELEASE release_ready=true requires passed release-gate checks
EVIDENCE evidence criterion_or_check_ref must reference valid criterion or check ID
COUNTS assessment counts must be ≥ 0
