Supervisor & Delegate Contract

Typed, bounded delegation between a supervisor agent and a delegate agent. Every handoff is explicit: request, context slice, constraints, assignment, structured return, and supervisor review. No ad-hoc child agents, no free-text handoffs.

Validation Rules
NON-BLANK delegation_id, supervisor_agent_id, objective, assignment_id, context_slice_id, constraint_id, selection_id, return_id, delegate_agent_id, review_id, reviewer_agent_id, envelope_id
ACCEPTANCE acceptance_criteria must be non-empty
CONTEXT context_slice needs at least one ref unless self_contained=True
CONFIDENCE selection_confidence and return confidence must be 0.0-1.0
RUNNING RUNNING status requires selected_delegate_id
RETURNED RETURNED status requires DelegationReturnRecord
REVIEWED ACCEPTED/REJECTED/REROUTED require SupervisorReviewRecord
CONSISTENCY return_id must match between return and review records
ACCEPT accept disposition needs accepted_output_refs or integration_notes
RETRY retry_same_delegate requires retry_reason
REROUTE reroute_to_other_delegate requires reroute_reason
NESTED must_not_delegate_further=True blocks nested delegation
