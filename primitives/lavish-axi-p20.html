Flow
INPUT GUARDRAIL InputPayload (user / retrieved / tool / web) └→ GuardrailEvaluation ─→ ALLOW / SANITIZE / REJECT / ESCALATE / REQUIRE_CONFIRMATION └→ GuardrailDecision (final_action + rationale + audit flag) OUTPUT GUARDRAIL OutputPayload (intended_for_user / intended_for_tool) └→ GuardrailEvaluation ─→ ALLOW ─→ deliver ├→ SANITIZE ─→ SanitizedOutput ─→ deliver ├→ REJECT ─→ block + audit └→ ESCALATE ─→ human review
Validation Rules
NON-EMPTY policy_id, rule_id, payload_id, evaluation_id, decision_id
NON-EMPTY content, sanitized_content, rationale, evidence_type, detail
CONSISTENT blocked=True → triggered_rules not empty
CONSISTENT action=REJECT → triggered_rules not empty
OUTPUT at least one of intended_for_user or intended_for_tool must be True
SCHEMA allowed_output_formats required when require_schema_validation=True
