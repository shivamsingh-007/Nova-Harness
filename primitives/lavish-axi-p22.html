Enforcement Flow
ToolAccessRequest arrives at runtime ├→ Check grants for agent_identity + tool_id + verb + environment ├→ Check explicit rules for overrides ├→ Evaluate scope constraints └→ Return AuthorizationDecision ├→ ALLOW → execute tool ├→ DENY → block (default if no match) └→ REQUIRE_APPROVAL → route to P21
Validation Rules
NON-EMPTY matrix_id, tool_id, tool_name, grant_id, rule_id, request_id, decision_id, agent_identity
VERBS supported_verbs not empty on tools • allowed_verbs not empty on grants
DESTRUCTIVE destructive=True forces risk_level HIGH or CRITICAL • DELETE/SEND require approval_required=True
ENVIRONMENTS environments must not be empty on grants and rules
DECISIONS non-DENY decisions must have matched_rule_ids • REQUIRE_APPROVAL needs approval_required=True
