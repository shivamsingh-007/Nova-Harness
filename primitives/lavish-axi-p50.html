Validation Rules
NON-BLANK role_id, name, profile_ids, assignment_id, agent_id, envelope_id
SUPERVISORY supervisory autonomy only for manager-like roles
ACTIVE active roles must have capability and constraint profiles
DELEGATE can_delegate roles require explicit delegation boundaries
MODIFY can_modify_artifacts requires verification or approval boundary
CONFLICT forbidden tools cannot appear in allowed_tools
ASSIGN assignment records must reference an active role
