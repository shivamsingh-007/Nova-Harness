Validation Rules
NON-BLANK handoff_id, from_agent_id, objective, packet/profile/ownership/return/review/envelope ids
TARGET either to_agent_id or target_role must be set
CRITERIA acceptance_criteria must be non-empty
OWNERSHIP full_transfer/temporary_execution require writer_agent_id
READ-ONLY read_only_assist and review_only must NOT set writer_agent_id
CONFIDENCE bounded [0.0, 1.0]
DELEGATE must_not_delegate_further constraint preserved through return
